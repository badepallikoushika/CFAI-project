# Intelligent Energy-Driven Productivity Planner

## Project Overview

The Intelligent Energy-Driven Productivity Planner is an AI-inspired web application designed to help users plan their work more effectively based on their energy levels rather than only time management.

The system analyzes factors such as sleep duration, stress level, mood, and work/study hours to estimate productivity levels and provide intelligent task recommendations.

---

## Problem Statement

Many students and professionals schedule tasks without considering their mental and physical energy levels. This often leads to reduced productivity, stress, and burnout.

This project aims to provide smarter productivity planning through intelligent decision-making and energy-based task recommendations.
